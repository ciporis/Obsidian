Implicit casting - неявное преобразование, совершаемое компилятором
explicit casting - явное преобразование, совершаемое разработчиком

Сравнение чисел с плавающей точкой
```
static void RightMain()
{
    double x = 1.0 / 10;
    double sum = 0;
    for (int i = 0; i < 10; i++)
        sum += x;
    Console.WriteLine(Math.Abs(sum - 1) < 1e-9);
}
```
В конце sum должна равняться 1, но она равна 0.99999999999999989. Таким образом нужно вычесть из нее предполагаемое число и сравнить его модуль с каким-то маленьким числом. Я так понимаю 1e-9 всегда будет работать корректно.
