# ref
Ключевое слово ref говорит компилятору, что объект уже проинициализирован до входа в метод.
Метод может читать и перезаписывать объект

# out
Ключевое слово out говорит компилятору, что объект будет инициализирован в теле метода.
Метод должен инициализировать объект до возвращения

# yield return
Сообщает компилятору, что нужно вернуть значение, когда итератор заканчивает работу

# yield break
Сообщает компилятору о прерывании перечисления раньше времени
