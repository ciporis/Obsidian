Time algorythm complexity - функция, показывающая точную верхнюю границу количества элементарных операций, необходимых для завершения работы алгоритма в зависимости от количества символов во входе.

Space algorythm complexity - аналогичная оценка для дополнительной памяти, необходимой для анализа входа. Память, использующаяся для хранения входа не учитывается.
