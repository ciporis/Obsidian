*Core/Meta* - ключевая механика игры(Meta - прокачка)

*Прогрессия* - всё что связано с тем как игра развивается для игрока(ощущение)

*Диегетический(Diegetic)*- когда элементы игры описываются внутри мира(checkpoints - магия, respawn - проклятье)

*Грокнуть* - понять игру полностью

*Pacing* - интенсивность событий во времени(экшн - передышка - экшн)

Кривая сложности - описание сложности игры во времени

*Gray boxing/Black out/White boxing* - прототипирование уровня без использования дорогостоящей графики

*Polishing* - полирование, процесс финализации игры(добавление всех звуков, анимаций). 
Может занимать 80% времени разработки

*PIZZAZZ* -  промежуточный polishing

*Skinner Box* - набор игровых механик для вызывания у игрока зависимости
Исследования проводил доктор Скиннер

*Телеграфирование* - все действия игры для предупреждения игроки о 
происходящем(вспышки, подсказки и т.д)
