Sphere collider
	Edit collider - только для трех базовых коллайдеров
	Is trigger - превращает объект в зону, попадание в которую могут обработать другие объекты
	Provides contacts - нет почему-то в моей версии движка, но при включении движок сам записывает все события, происходящие с коллайдером, при выключении записываются события, заданные разработчиком
	Material - определяет всякие физические параметры для объекта
	Center - определяет центр коллайдера относительно объекта
	Radius (для сферы) - определяет радиус коллайдера
Terrain collider
	Terrain data - хранит данные о terrain collider
	Enable tree colliders - включение и  выключение коллайдеров для деревьев
