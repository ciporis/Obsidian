### Range
Определяет промежуток, в котором может находиться значение поля

### Min
Определяет минимальное значение, которое может принимать поле

### MultiLine
Определяет количество строк, которое нужно для появления скролл бара

### TextArea
Определяет размер поля для ввода текста

### ToolTip
Оставляет подсказку при наведении

### Header
Оставляет заголовок

### Space
Регулирует расстояние между полями

### Delayed
Запрашивает разрешение на изменение значения в поле

### Serializable
Отображает класс в инспекторе целиком 

### NonSerializable
Отмечает public поля, которые не нужно отображать в инспекторе

### HideInInspector
Отмечает public поля, которые можно serialize, но их не видно в инспекторе

### ContextMenu
Применяется к методам и позволяет вызывать их в редакторе 

### RequiredComponent
Указывает компонент, необходимый для класса

### SelectionBase
При клике на дочерний объект будет выбираться вся совокупность
