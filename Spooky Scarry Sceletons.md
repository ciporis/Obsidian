# Дано:
1) n видео файлов в папке videos
2) n аудио файлов в папке music
# Задача:
Скрипт делает запрос в нейросеть, получает от нее текст страшной истории. Далее делает запрос в другую нейросеть и получает от нее аудио файл с озвучкой этого текста. Далее получается случайный видео файл из videos и случайный аудио файл из music.
Видео файл из videos и аудио файл из music обрезаются так, чтобы их длительность составляла столько же сколько длительность файлы озвучки текста. Так же у файла из videos должен быть убран звук. В итоге в папке results сохраняется видео ролик с музыкой, фоновым видео и аудио с рассказом диктора.
