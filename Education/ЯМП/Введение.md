### Редактор SciNotes
Используется для записи программ на языке SciLab

##### Цикл for
Используется для составления циклов с определенным количеством итераций
Структура:

for i = a : h : b
	операции
end

a, b - пределы изменения переменной
h - шаг

По умолчанию h = 1, в частности можно указать h = -1

##### Условный оператор if
Структура:

if <условие> then <операции>
elseif <условие> then <операции>
else <операции>
end

##### Цикл while

Структура:

while <условие>
	<действия>
end

### Решение задач с массивами
1. Одномерные массивы
- Ввод массивов
```
A = [1, 3, 4, 5, 6]
```
Нумерация элементов начинается с 1
- Действия над одномерными массивами

| Scilab    | Функция                                    |
| --------- | ------------------------------------------ |
| lenght(A) | Возвращает длину массива(кол-во элементов) |
| max(A)    | Возвращает наибольшее значение             |
| min(A)    | Возвращает минимальное значение            |
| sum(A)    | Возвращает сумму                           |
| prod(A)   | Возвращает произведение всех элементов     |
- Примеры задач
1) Дан одномерный массив, состоящий из натуральных чисел. Найти сумму чисел массива, больших трех и меньших десяти

2) Найти произведение чётных чисел

3) Найти количество чисел, не кратных трём


Решения в виде функций в редакторе:

```
() - входные параметры
[] - выходные параметры

1) function[S] = f(A)
   S = 0
   n = length(A)
   for i = 1 : n
	   if A(i) > 3 & A[i] < 10 then S = S + A(i)
	   end
   endfunction
```

Для запуска необходимо выбрать в меню Инструменты > Сохранить и выполнить
Затем в рабочем окне вызвать функцию f(A), появятся выходные параметры либо сообщение об ошибке.

2. Двумерные массивы
- Ввод массивов
A = [1, 2, 3, 4; 8, 2, 5, 2; 6, 7, 0]
- Функции

| Scilab            | Функция                                                                                               |
| ----------------- | ----------------------------------------------------------------------------------------------------- |
| [m, n] == size(A) | Переменным n и m присваиваются значение, равные количеству строк и количеству столбцов соответственно |
| det(A)            | Определитель матрицы                                                                                  |
| inv(A)            | Обратная матрицы                                                                                      |

- Примеры задач
1) Дана квадратная матрица с натуральными числами, найти сумму элементов главной диагонали.
2) Найти произведение элементов побочной диагонали, больших определенного числа a.
3) Найти количество нечётных элементов, расположенных под главной диагональю.

```
1) function[s] = f(A)
	   [m, n] = size(A)
	   s = 0
	   for i = 1 : m
	   for j = 1 : n
		   if i == j then s = s + A(i, j)
		   end
	   end
	   end
   endfunction
```

```
2) function[p] = f(A)
	   [m, n] = size(A)
	   p = 1
	   a = input("Введите число a")
	   
	   for i = 1 : m
	   for j = 1 : n
		   if i + j == m + 1 & A(i, j) > a then p = p * A(i, j)
		   end
	   end
	   end
   endfunction
```

```
3) function[N] = f(A)
	   [m, n] = size(A)
	   N = 1
	   
	   for i = 1 : m
	   for j = 1 : n
		   if i > j & modulo(A(i, j), 2) == 0 then N = N + 1
		   end
	   end
	   end
   endfunction 
```

Если после then требуется выполнить несколько действий они записываются через запятую.
